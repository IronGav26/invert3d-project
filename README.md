# _what is this?_
invert3d is an isometric puzzle game made for the 2nd 1-BIT jam on itch.io.

# _controls_
[CONTROL: ACTION]
WASD: move around the map
E: invert colors (maybe you'll find something)
F: return colors to normal
left click: interact, attack
escape: menu
